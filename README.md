## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Create a new Oauth App to get a client ID  [https://github.com/settings/developers](https://github.com/settings/developers)
2. Clone the repo
   ```sh
   git clone https://github.com/kawtar-Zinoun/Zinoun_Kawtar-Coding-Challenge-Frontend-VueJs
   ```
3. Clone the GateKeeper repo to  send POST requests to the Github Api
    ```sh
   git clone https://github.com/kawtar-Zinoun/gatekeeper
   ```
   
4. Install NPM packages
   ```sh
   npm install
   ```
5. In the gateKeeper main folder, Enter your client ID and your client Secrent provided by Github in `config.json`
   ```js
   "oauth_client_id": "YOUR_CLIENT_ID",
  "oauth_client_secret": "YOUR_CLIENT_SECRET",
   ```
6. Run GateKeeper
   ```sh
   npm start 
   ```
6. Run the coding challenge
   ```sh
   npm serve
   ```
